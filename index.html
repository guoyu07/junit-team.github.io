JUnit

JUnit - the programmer-oriented testing framework for Java.

JUnit 4

JUnit 4 is the current stable version of JUnit. For more information, please visit the JUnit 4 website.

JUnit 5

JUnit 5 is the next generation of JUnit. The goal is to create an up-to-date foundation for developer-side testing on the JVM. This includes focusing on Java 8 and above, as well as enabling many different styles of testing. For more information, please refer to the User Guide or the Source Code on GitHub.
